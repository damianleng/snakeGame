# Multiplayer Snake Game

Welcome to the Multiplayer Snake Game! This Java game allows two players to play the classic snake game. It features a scoreboard to keep track of each player's score, and it includes sound effects to enhance the gaming experience.

## Features

- **Multiplayer**: Two players can compete against each other on the same screen, controlling separate snakes.
- **Scoreboard**: The game displays a scoreboard to keep track of each player's score.
- **Sound Effects**: Sound effects are incorporated to provide an immersive gaming experience.

## Prerequisites

Before running the game, make sure you have the following prerequisites:

- Java Development Kit (JDK) installed on your machine.

## Getting Started

To get started with the Multiplayer Snake Game, follow these steps:

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/your-username/multiplayer-snake-game.git
   ```

2. Navigate to the project directory:

   ```bash
   cd multiplayer-snake-game
   ```

3. Compile the Java code:

   ```bash
   javac SnakeGame.java
   ```

4. Run the game:

   ```bash
   java SnakeGame
   ```

5. Follow the on-screen instructions to play the game. Each player controls their snake using separate controls (e.g., arrow keys and WASD).

## Controls

Player 1:
- Up: Arrow Up
- Down: Arrow Down
- Left: Arrow Left
- Right: Arrow Right

Player 2:
- Up: W
- Down: S
- Left: A
- Right: D
